# Ecosystem

Out of the box, Ollie contains a CSS framework also made by Virtually(Creative) called, [Allegretto]().

A light-weight frameworks to help with making wireframe layouts easy with vanilla CSS.

You can easily swap it for whatever front-end framework you wish. Its referenced in the `<head>` tag in `/src/index.ejs`.

```html
  <!-- Load Allegretto from CDN -->
  <link rel="stylesheet" href="https://unpkg.com/allegretto@latest/dist/retto.min.css" type="text/css" media="all">
```
